# plantuml-to-mermaid

Convert plantuml to mermaid by shell scripts.


## How to use
You need `main.sh` to run.

``` sh
$ bash main.sh <target_markdown_file>
# specify output filename
$ bash main.sh <target_markdown_file> -o <output_markdown_file>
```
